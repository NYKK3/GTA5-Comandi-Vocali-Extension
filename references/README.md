# Cartella References

Questa cartella deve contenere le DLL di riferimento necessarie per compilare il progetto.

## DLL Richieste

Copia le seguenti DLL in questa cartella:

1. `GrammarPolice.dll`
2. `LSPD First Response.dll`
3. `MechanicBackup.dll`
4. `RagePluginHookSDK.dll`
5. `StopThePed.dll`
6. `UltimateBackup.dll`

## Dove trovare le DLL

Queste DLL sono disponibili nei rispettivi repository GitHub:

- **GrammarPolice**: [lcpdfr](https://www.lcpdfr.com/downloads/gta5mods/scripts/28003-grammarpolice/)
- **LSPD First Response**: Questo dll si trova nella cartella: Grand Theft Auto V > plugins
- **RagePluginHookSDK**: Questo dll si trova nella cartella: Grand Theft Auto V > SDK
- **UltimateBackup**: [bejoijo](https://www.bejoijo.com/post/ultimate-backup)
- **StopThePed**: [bejoijo](https://www.bejoijo.com/post/stop-the-ped)
- **MechanicBackup**: [lcpdfr](https://www.lcpdfr.com/downloads/gta5mods/scripts/29789-mechanicbackupplugin/)

## Istruzioni

1. Scarica tutte le DLL necessarie dai repository sopra elencati
2. Copia le DLL in questa cartella `references/`
3. Compila il progetto con Visual Studio

## Note

- Assicurati di usare versioni compatibili delle DLL
- Le DLL devono essere nella stessa cartella `references/` per funzionare correttamente
